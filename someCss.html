What's CSS?

CSS stands for Cascading Style Sheets.
It is used to create a look and feel for a web page, describing how how HTML elements will be displayed.
CSS consists of properties and values that are applied to an HTML Tags.


What's a property?

The properties state what sorts of look and feel you want to apply.
For example, we use the property 'background-color' to apply a background code inside a tag.


What's a value?

The values describe what to do with the property.
For example, we can enter background-color:red to say that we want the background color to be red.
We use a colon (:) to separate the property from the value.

How do you say "this property has this value?

Like I've described it above, we use a colon (:) to link the property and the value.
If we have more than one property and value for a tag, we separate the properties with a semin-colon (;)
